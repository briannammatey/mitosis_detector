## Languages and Tools
We used Google Colab, Python, Pytorch, scikit-learn
## Description
The aim for this project is to create a Machine Learning Mitosis Detector, a computer vision project aimed at automatically identifying and classifying mitotic cells in breast cancer images. By training on labeled breast cancer image datasets, the model learns to distinguish between mitotic and non-mitotic cells.

## Our Approach
We split up our group so that three will use the midnight model, and two will use the deeplabv3 model
We used techniques like Data Augmentation, early stopping, Adam optimizer, and learning rate scheduling to improve accuracy.

## Results
We successfully came to an accuracy of 84% of 26,000+ images over 50 epochs
